## 图片要求

1. 格式要求：
   - 照片类图片使用jpg格式
   - 图标类图片使用png或svg格式
   - 网站图标使用ico格式

2. 尺寸要求：
   - 头像：建议尺寸 200x200 像素
   - 封面图：建议尺寸 1200x630 像素
   - 背景图：建议尺寸 1920x1080 像素
   - 作品展示图：建议尺寸 800x600 像素

3. 风格要求：
   - 保持专业商务风格
   - 与产品、敏捷、技术主题相关
   - 整体视觉统一性
   - 图片清晰度高
   - 文件大小适中（建议单张不超过500KB）

## 注意事项

1. 所有图片资源需要自行准备或从合法渠道获取
2. 注意图片版权问题，建议使用原创图片或获得授权的图片
3. 图片命名使用小写字母，单词间用连字符(-)连接
4. 定期检查和更新图片资源，确保链接有效性
